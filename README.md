# Как настроить

1. Склонировать репозиторий

```
git clone https://github.com/McConderEz/KafkaProduceConsumeExample.git
```
2. Настроить подключение в compose.yaml и appsettings.json.

3. Поднять kafka и kafka ui в докер. Для этого в корне проекта выполнить следующую команду

```
docker-compose up -d
```

4. Запустить проекты Producer и Consumer.
